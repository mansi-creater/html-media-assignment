List of HTML Tags

  1. Heading (h1 to h6) :A HTML heading or HTML h tag can be defined as a title or a subtitle which you want to display on the webpage.There are six different HTML headings which are defined with the h1 to h6 tags.
  2. Paragraph (p) :HTML paragraph or HTML p tag is used to define a paragraph in a webpage. An HTML p tag indicates starting of new paragraph.
  3. Div Tag (div): The HTML DIV tag is used to group the large section of HTML elements together.
  4. Table Tag (table) :HTML table tag is used to display data in tabular form (row * column). There can be many columns in a row.
  5. Ordered List (ol) :HTML Ordered List or Numbered List displays elements in numbered format. The HTML ol tag is used for ordered list.
  6. Unordered Tag (ul) :HTML Unordered List or Bulleted List displays elements in bulleted format . The HTML ul tag is used for the unordered list.
  7. Italic Tag (i) :HTML b tag is used to display the written text in bold format. It is strictly a presentational element.
  8. Underline Tag (u)HTML U> tag is used to define a span of inline text with a non-textual annotation. It rendered as an solid underlined text.
